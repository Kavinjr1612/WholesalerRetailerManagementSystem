# Login Images

Place the following images in this folder:

1. `logo.png` - The logo used for both admin and retailer login pages
2. `adminbackground.jpg` - Background image for the admin login page
3. `retailerbackground.jpg` - Background image for the retailer login page

Image Guidelines:
- Logo: Should be square, recommended size 200x200px
- Background images: Recommended minimum size 1920x1080px
- Use high-quality, optimized images for better loading performance
- Supported formats: JPG, PNG
